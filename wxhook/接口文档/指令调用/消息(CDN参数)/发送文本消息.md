
**请求**

 ``` 
 {
  "type": 11237,
  "data": {
    "to_wxid": "filehelper",
    "content": "new msg interface"
  }
} 

 ```



> to_wxid 接收者的微信id
> content 文本消息内容


**返回**
 ``` 
 {
  "data": {
    "baseResponse": {
      "ret": 0
    },
    "count": 1,
    "msgResponseList": [
      {
        "clientMsgId": 1658910443,
        "createTime": 1658910443,
        "msgId": 0,
        "newMsgId": "267817xxxxxxxxxx2",
        "ret": 0,
        "serverTime": 1658910445,
        "toUserName": {
          "string": "filehelper"
        },
        "type": 1
      }
    ]
  },
  "type": 11237
} 

 ```
> newMsgId 消息唯一Id, 可用于撤回
