 **JSON示例**

``` 
{
   "data" : {
      "at_user_list" : ["wxid_xxxxxxx", "wxid_xxxxxxxx"],
      "from_wxid" : "xxxxxxxxx",
      "msg" : "你好，世界",
      "room_wxid" : "xxxxxx@chatroom",
      "to_wxid" : "wxid_xxxxxxxx",
      "wx_type" : 1
   },
   "type" : 11046
}

```


`from_wxid也包含是自己的情况，使用时注意判断，避免循环发送`

`room_wxid 判断这个字段区分是否为群聊消息`
