`from_wxid也包含是自己的情况，使用时注意判断，避免循环发送`

`room_wxid 判断这个字段区分是否为群聊消息`

 **JSON示例**

``` 
{
   "data" : {
      "from_wxid" : "xxxxxxx",
      "raw_msg" : "xxxxx",
      "room_wxid" : "",
      "to_wxid" : "wxid_xxxxxx",
      "wx_type" : 10000
   },
   "type" : 11058
}
```
