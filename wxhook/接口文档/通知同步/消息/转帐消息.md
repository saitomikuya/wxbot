`from_wxid也包含是自己的情况，使用时注意判断，避免循环发送`

`room_wxid 判断这个字段区分是否为群聊消息`

 **JSON示例**

``` 
{
   "data" : {
      "from_wxid" : "xxxxxxx",
      "raw_msg" : "<?xml version=\"1.0\"?>\n<msg>xxxxxx</msg>",
      "to_wxid" : "wxid_xxxxxx",
      "wx_type" : 49,
	  "wx_sub_type" : 2000
   },
   "type" : 11057
}
```
`raw_msg包含转账信息，可以发送接收转账接口，也可以向接口设置自动接收转帐`
