`from_wxid也包含是自己的情况，使用时注意判断，避免循环发送`

`room_wxid 判断这个字段区分是否为群聊消息`

 **JSON示例**

``` 
{
   "data" : {
      "from_wxid" : "xxxxxxx",
      "raw_msg" : "<?xml version=\"1.0\"?>\n<msg>xxxxxx</msg>",
      "room_wxid" : "",
      "to_wxid" : "wxid_xxxxxx",
      "wx_type" : 37
   },
   "type" : 11049
}
```
`raw_msg包含encryptusername,ticket信息，使用encryptusername,ticket可以直接调用同意好友请求接口`
