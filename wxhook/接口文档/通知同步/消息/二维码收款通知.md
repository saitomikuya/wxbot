**Data JSON参数：** 

|参数|类型|描述|
|:-------|:-------|:-------|
| wxid | string| 付款者的wxid |
| nickname | string| 付款者的昵称 |
| fee | string| 支付金额 ( 单位分) |
| orderno | string|订单号|
| timestamp | number | 支付时间  |


 **JSON示例**

``` 
{
   "data" : {
      "wxid" : "xxxxxxx",
      "nickname" : "xxxxxx",
      "fee" : "100",
      "feetype" : "2",
      "orderno" : "10011111111",
	  "timestamp" : "1509203021"
   },
   "type" : 11095
}
```
