接口将通过json格式进行通信，如
```json
{
	"type": 10,
	"data": {},
	"trace": 1
}
```

> type 为指令类型值，必须是整数
> data 为指令附加内容
> trace 由调用方传递过来，接口会原样返回，可用于指令同步。（ps: 所有接口都有）

**大客户HOOK版本与普通HOOK版本的区别**

- 所有接口都有返回值
- 所有接口请求与返回一一对应， 很大程序上降低了开发难度，可以将异步转同步
- 聊天媒体文件可以默认不下载，自行安需下载
- 内存低耗，全部走的底层协议，没有经过UI
- 发送媒体文件(图片，视频，文件，语音)接口走的cdn参数发送，高效不占内存
- 支持朋友圈功能（可以看全部，看人个，点赞，评论，转发朋友圈，发朋友圈（图，文））
- 更全的群管理功能和好友功能
- 支持企业微信群和联系人管理
- 支持标签功能
- 支持全部类型的CDN文件的上传和下载
- 为了兼容旧版本，标记为旧的接口，不支持trace
- 支持视频号全部功能
- 支持视频号直播全部功能
